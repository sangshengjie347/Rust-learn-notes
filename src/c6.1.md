# 堆、栈和指针

## 栈
栈以放入值的顺序存储值，并以相反的顺序的取出值。     --`后进先出`
栈上的数据必须是`固定大小`的。

## 堆
当存储的`数据大小未知或者可变`的则要放在堆上。

## 对比
入栈要比在堆上分配内存要快，访问栈上的数据同样比访问堆上的数据要快。

## 指针
指针大小已知且固定的，所以它存储在栈上。

操作系统在堆的某处找到一块足够大的空位，把它标记为已使用，并返回一个表示该位置地址的指针。
```rust
fn main() {
    let s = String::from("hello");
}
```
